# Claude-introduction-and-usage-guide
The only ultimate Claude setup and usage guide that you will ever need!

Claude Desktop Installation
---
Claude chatbot has two ways to be accessed. 
- [Website](Claude.com)
- Desktop app - download from [Claude.com](https://claude.com/)

The desktop is built off of [Electron](https://electronjs.org), so the desktop app is not necessarily needed.
___

Settings
---

Head to the settings under your profile in the bottom left corner. Head to the file [Settings.md](https://github.com/asmbelly/Claude-introduction-and-usage-guide/blob/main/Settings.md) to see my exact settings or how to personalize your settings.

___

Artifacts
---

Claude Artifacts are interactive, standalone workspace outputs generated by the AI that appear in a dedicated side panel next to your main chat. 

Instead of forcing you to copy-paste code or text, Artifacts create an environment where you can view, edit, run, and share your creations directly within the app

___

Skills
---

Claude Skills are reusable, modular instruction packages that teach Claude how to handle specific tasks, workflows, or roles.

Instead of having to copy-paste text and manually do all the hard labor, Claude skills aim to automake that workflow to provide a better useer experience.
